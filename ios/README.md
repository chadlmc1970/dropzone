# DropZone iOS App

Native iOS DJ mixing app built with SwiftUI and AVAudioEngine.

## Tech Stack
- SwiftUI (iOS 15+)
- AVAudioEngine
- Spotify iOS SDK
- MVVM Architecture

## Build
```bash
# Open in Xcode
open Package.swift

# Or build from CLI
swift build
```
